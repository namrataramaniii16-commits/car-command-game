# Car Command Game

A simple beginner-friendly Python project where the user controls a car using text commands.

## How It Works

The player can type commands to start or stop the car:

- `start` starts the car
- `stop` stops the car
- `help` shows the available commands
- `quit` exits the game

The program also checks whether the car is already started or already stopped.

## How to Run

Make sure Python is installed on your computer.

Run the game with:

```bash
python main.py
```

On some systems, you may need:

```bash
python3 main.py
```
